Fitur Sistem

Fitur lengkap untuk analisis risiko TI menggunakan COBIT 2019

1
Autentikasi & Keamanan

Login dengan session authentication, password hashing bcrypt, CSRF protection, dan SQL injection prevention.

2
Dashboard Analytics

Dashboard modern dengan statistic cards, grafik capability level, gap analysis, dan radar chart.

3
Framework COBIT 2019

Informasi lengkap domain DSS01 (Manage Operations) dan DSS05 (Manage Security Services).

4
Data Penilaian

CRUD responden, form penilaian dinamis DSS01 & DSS05 dengan skala 0-5, progress bar, dan validasi.

5
Hasil Analisis & Rekomendasi

Perhitungan capability level otomatis, gap analysis, rekomendasi berbasis gap, dan status kondisi.

6
Laporan & Export

Preview laporan lengkap, cetak PDF, dan export data ke format CSV.

Panduan Instalasi

Langkah-langkah menginstal sistem pada server lokal

Langkah 1: Copy Project
# Copy folder project ke htdocs atau www C:\xampp\htdocs\eraport-cobit
Langkah 2: Buat Database
-- Login ke MySQL mysql -u root -p -- Buat database CREATE DATABASE eraport_cobit CHARACTER SET utf8mb4;
Langkah 3: Import Database
-- Import file database.sql mysql -u root -p eraport_cobit < database.sql
Langkah 4: Konfigurasi
// Edit config/database.php define('DB_HOST', 'localhost'); define('DB_USERNAME', 'root'); define('DB_PASSWORD', ''); define('DB_NAME', 'eraport_cobit'); define('BASE_URL', 'http://localhost/eraport-cobit');
Langkah 5: Akses Aplikasi
-- Buka browser http://localhost/eraport-cobit -- Login default Username: admin Password: password
Catatan: Sistem ini memerlukan server PHP dan MySQL. Gunakan XAMPP, WAMP, atau LAMP untuk menjalankan sistem.
